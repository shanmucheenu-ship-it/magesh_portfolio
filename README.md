# Portfolio Website

A modern, interactive portfolio website built with React, Tailwind CSS, and Framer Motion.

## Features

- ✨ Smooth animations with Framer Motion
- 🎨 Dark theme with gradient accents
- 📱 Fully responsive design
- 🖱️ Custom cursor effect
- 🔄 Animated skill bars
- 📂 Filterable project gallery
- 📧 Working contact form
- ⚡ Optimized performance

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React
- React Intersection Observer

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Customization

1. Update personal info in components
2. Replace emoji placeholders with actual images
3. Add your social links
4. Update project data in `Projects.jsx`
5. Customize colors in `tailwind.config.js`
